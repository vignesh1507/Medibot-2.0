# MEDIBOT_REM

## Project Overview
MEDIBOT_REM is a health-focused AI chat application designed to provide personalized responses based on user interactions. The application leverages AI models to assist users with health-related inquiries and offers a user-friendly interface for seamless communication.

## Project Structure
```
MEDIBOT_REM
├── app
│   ├── chat
│   │   └── page.tsx        # Main chat functionality, manages chat sessions and AI responses
│   └── layout.tsx          # Layout structure for the application
├── components               # Reusable UI components
├── lib                      # Utility functions and libraries
├── package.json             # npm configuration file
└── README.md                # Project documentation
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd MEDIBOT_REM
   ```

2. **Install Dependencies**
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Run the Application**
   To start the development server, use:
   ```bash
   npm run dev
   ```

4. **Access the Application**
   Open your browser and navigate to `http://localhost:3000` to access the chat application.

## Usage
- Users can initiate a chat session by typing their health-related inquiries.
- The AI assistant will respond based on the user's previous interactions and the context of the conversation.
- Users can upload files for analysis, and the AI will provide insights based on the content.
